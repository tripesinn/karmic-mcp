## What it does

A Swiss Ephemeris-backed MCP server providing **precision astrological analysis** for use with on-device Gemma-4-E4B-it. Three tools exposed:

| Tool | Purpose |
|------|---------|
| `get_natal_chart` | Birth chart positions (Sun, Moon, Ascendant, planets, nodes) |
| `get_transits_today` | Current planetary aspects for a given natal chart |
| `get_doctrine_reading` | Synthetic evolutionary doctrine interpretation |

Built specifically to stay within Gemma 4B's context window constraints — bite-sized responses, no bloat.

## 🚀 Quick Start for Edge Gallery users

1. Open **Edge Gallery** on your Pixel device
2. Go to **Settings → MCP Servers**
3. Tap **Add custom server**
4. Enter URL: `http://34.163.125.49:8000`
5. Test by asking Gemma:

## 🤖 Fun fact for the local-AI community

The **entire codebase** (FastAPI server, Pydantic models, deployment scripts, this very README, and the systemd service config) was generated by **Gemma-4-E4B-it running locally** on a Mac Mini M4 (16 GB RAM) via oMLX — zero cloud LLM calls during development. Proof that small local models can ship production services, not just chat demos 🚀
